## 与 v3.0 节点模块的集成

Services 模块与 v3.0 版本的 Nodes 模块紧密集成，提供以下关键能力：

**LLM 服务集成**：
- 为意图分类节点提供结构化输出能力
- 为决策节点提供治疗方案生成能力
- 为审核节点提供质量审查能力
- 支持流式输出和思考模式

**文档转换器集成**：
- 为前端提供病历文档结构化能力
- 输出格式直接兼容可视化组件
- 支持隐私脱敏保护

**联网搜索集成**：
- 为知识检索节点提供实时搜索能力
- 支持临床证据、药物信息、指南更新等多场景搜索
